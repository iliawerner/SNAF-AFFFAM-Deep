# SNAF & Deep Framework — Viral Prediction Calculator

Статистически обоснованный калькулятор для оценки вирального потенциала видео на YouTube-канале Design Lovers.

## Структура репозитория

| Файл | Описание |
|---|---|
| `index.html` | Калькулятор (открыть в браузере или через GitHub Pages) |
| `AUDIT_REPORT.md` | Полный отчёт аудита: найденные ошибки, методология, новая модель |
| `model_params.json` | Параметры регрессионных моделей (A, B, C) в машиночитаемом формате |
| `data/sheet_fnf.csv` | Исходные данные: 33 видео с оценками по 9 факторам |

## Формула (Модель B)

```
ln(Views) = 8.81 + 0.198 × Fam_Rec + 0.137 × Affect_Now − 0.143 × Affect_Psych − 0.071 × Fam_Neg
```

**Adj. R² = 0.36** | **LOO CV RMSE = 1.13** | **n = 33**

## Как использовать

Откройте `index.html` в браузере. Введите оценки по 4 ключевым факторам (0–10), и калькулятор покажет прогнозируемый диапазон просмотров.

## Подробности

Полное описание аудита, найденных ошибок в предыдущих моделях и методологии построения новой модели — в файле [AUDIT_REPORT.md](./AUDIT_REPORT.md).
